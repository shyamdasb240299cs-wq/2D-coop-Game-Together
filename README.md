# 2D Coop Game Together

Welcome to **2D Coop Game Together**, a cooperative 2D multiplayer platformer!

> **Note:** This repository is dedicated to distributing the playable builds of the game. The source code is kept private.

## Features
- Cooperative 2D platforming action
- Multiplayer networking support
- Custom level design and elements

## How to Play

### Android
1. Navigate to the `apk V1` folder in this repository.
2. Download one of the `.apk` files (e.g., `2D COOP V1.apk`).
3. Transfer the APK to your Android device.
4. Open the file on your device to install the game (you may need to allow "Install from unknown sources" in your device settings).
5. Launch the game and have fun!

### Windows
1. Download the `Test Build` folder to your computer.
2. Extract it if downloaded as a zip from GitHub.
3. Ensure that the game executable (`Together.exe` or `2d coop.exe`) and its associated `_Data` folder are kept together in the same directory.
4. Double-click the executable to start playing!

### Web
1. Navigate to the `Web Builds` folder.
2. These files can be hosted on any standard web server to play the game directly in your browser.
